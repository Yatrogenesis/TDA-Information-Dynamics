Initial release: Topological Data Analysis for Phase Transition Detection

## Features
- Persistent Homology analysis of 2D Lennard-Jones systems
- CUSUM-based early warning detection
- Statistical validation across N=144, 400, 900, 1600 particles
- 100% precursor detection rate for N>=900

## Key Finding
Topological instability (H1 entropy changes) statistically precedes metric ordering during crystallization.

- Python simulation scripts (v6-v9 series)
- Publication-quality figures (600 dpi)

- Supplementary material
